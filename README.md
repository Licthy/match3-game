# Match 3 Game

一个简单的消消乐游戏，使用纯 HTML/CSS/JavaScript 实现。

## 怎么玩

1. 点击两个相邻的宝石进行交换
2. 连成3个或以上相同宝石即可消除
3. 消除越多，分数越高
4. 步数有限，挑战最高分！

## 在线游玩

访问 GitHub Pages 链接（启用后）：
https://your-username.github.io/match3-game/

## 本地运行

双击 `index.html` 文件即可在浏览器中打开。

## 技术

- HTML5
- CSS3 (渐变、动画)
- JavaScript (ES6+)

## 特性

- 8x8 网格
- 6 种不同颜色的宝石
- 平滑的消除动画
- 分数和步数统计
- 自动填充新宝石
- 连击检测
